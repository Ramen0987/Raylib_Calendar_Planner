# 🗓️ Planner App

A lightweight task management app built in C++ using the raylib graphics library. Designed with simplicity and functionality in mind, this planner lets users create, view, and delete tasks—no distractions, just streamlined productivity.

---

## 🚀 Features

- 📌 Add and display tasks with intuitive navigation
- ✅ Delete completed or unwanted tasks
- 💾 Persistent task saving between sessions
- 🎨 Built with raylib for a clean graphical interface
- 🧠 Minimal design inspired by focus-driven workflows

---

## 🛠️ Tech Stack

- **Language:** C++
- **Graphics Library:** [raylib](https://www.raylib.com/)
- **Development Environment:** Visual Studio
